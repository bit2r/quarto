1 사전 준비

1.1 데이터 선택

항상 모든 사람이들이 말하는데….

펭귄은 귀요미!

… 이제부터 펭귄 데이터로 즐거운 데이터 사이언스를 시작해봅시다.

Note

팔머펭귄(palmerpenguins) 데이터셋은 기계학습과 데이터 사이언스에 흔히 사용되는 붓꽃(iris) 데이터를 대체하기 위해 제작된 것입니다.

팔머펭귄 데이터셋에 대한 자세한 사항은 Allison Horst’s Github page. 웹사이트를 참조합니다.

데이터 수집에 대해 궁금하신 분은 클릭하여 추가 학습하세요

팔머펭귄 데이터는 Kristen Gorman 박사와 남국 팔머 관측소(Palmer Station, Antarctica LTER)에서 수집되어 공공에 이용가능하게 공개되었습니다.

{palmerpenguins} 패키지 (Horst et al., 2020)를 통해 R에서 손쉽게 이용가능하게 되었고 원데이터는 Gorman et al. (2014) 에서 최초 공개되었다.

1.2 데이터와 패키지 불러오기

먼저, 다음 작업을 순차적으로 수행한다:

  1. 패키지 적재하기(loading)
  2. 데이터 불러오기
  3. 데이터가 올바른 형식인지 확인한다.

“올바른 형식”이란, 데이터가 정돈되어(tidy) 요약통계량, 시각화, 모형개발에 올바른 형태로 준비되어 있음을 의미함.
다운로드하지 않고 데이터 사용하는 방법

palmerpenguins 데이터 패키지에 동일한 데이터를 바로 이용할 수 있음.

2 시각화

시각화 그래프 제작을 위한 몇가지 아이디어:

  • 펭귄 체중과 물갈퀴 길이
    • 전체 데이터를 대상으로
    • 혹은 각 펭귄 종별로 나눠서
    • 부가적으로 성별도 고려하여
  • 물칼퀴 길이와 부리 길이
  • 부리 길이(bill length)와 부리 깊이(bill depth)

2.1 펭귄 체중과 물갈퀴 길이

다음 그래프는 {palmerpenguins} 패키지 소품문에 제시된 것은 간략하게 작성한 것임.

2.1.1 전체 데이터

Code 
mass_flipper <- ggplot(data = penguins,
                       aes(x = flipper_length_mm,
                           y = body_mass_g)) +
  geom_point(size = 3,
             alpha = 0.8) +
  theme_minimal() +
  labs(title = "팔머 관측소 LTER 서식 펭귄 크기",
       subtitle = "Adelie, Chinstrap, Gentoo 펭귄에 대한 물갈퀴 길이와 체질량",
       x = "물갈퀴 길이(Flipper length) (mm)",
       y = "체질량(Body mass) (g)") +
  theme_minimal()

2.1.2 펭귄 종별로 나눠서 시각화

Code 
mass_flipper_species <- ggplot(data = penguins,
                       aes(x = flipper_length_mm,
                           y = body_mass_g)) +
  geom_point(aes(color = species,
                 shape = species),
             size = 3,
             alpha = 0.8) +
  theme_minimal() +
  scale_color_manual(values = c("darkorange","purple","cyan4")) +
  labs(title = "팔머 관측소 LTER 서식 펭귄 크기",
       subtitle = "Adelie, Chinstrap, Gentoo 펭귄에 대한 물갈퀴 길이와 체질량",
       x = "물갈퀴 길이(Flipper length) (mm)",
       y = "체질량(Body mass) (g)",
       color = "펭귄 종",
       shape = "펭귄 종") +
  theme_minimal()
